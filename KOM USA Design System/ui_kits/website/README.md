# KOM USA — Website UI Kit

Interactive click-through recreation of the KOM USA marketing website: Home, Services, About, and Contact.

No production codebase or screenshots were provided for this brand — this UI kit is an original interpretation built strictly from the brand guide's documented rules (warm-white layouts, soft green accents, real project photography, practical CTAs, square-ish corners). Photography is represented with labeled placeholder blocks; swap these for real project photos.

Open `index.html` and use the top tab bar to move between pages.

Files: `Photo.jsx` (photo placeholder), `Logo.jsx` (logo variant loader), `HomePage.jsx`, `ServicesPage.jsx`, `AboutPage.jsx`, `ContactPage.jsx` (with a working free-estimate form).
