# Security Policy

This repository powers the live [kom-usa.com](https://kom-usa.com) website, which collects customer contact and job details through Netlify Forms.

## Reporting a vulnerability

If you discover a security issue, please report it privately rather than opening a public issue. This includes exposed credentials, a vulnerability in the site or its dependencies, or a way to access customer data you shouldn't have.

Email **contact@kom-usa.com** with details and, if possible, steps to reproduce. We aim to acknowledge reports within 2 business days.

## Scope

This applies to the code in this repository and the production/preview deployments built from it. It does not cover third-party services (Netlify, Storyblok) directly — report those issues to the respective provider.
